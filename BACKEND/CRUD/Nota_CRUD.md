# Pasta CRUD
As funções CRUD serão armazenadas nesta pasta, para fins de organização, e as funções CRUD relacionadas a cada iten ficaram em um mesmo arquivo, a qual deve deve ser nomeado como ``CRUD_(nomeItem).js```, também para motivos de organização.

````javascript
/*Exemplo:
    CRUD/
      CRUD_usuarios.js --> Arquivo onde estarar as funções GET, POST, PUT e DELETE de usuario.
*/ 
```
