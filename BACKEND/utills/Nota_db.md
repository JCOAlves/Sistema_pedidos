# Dados da conexão do Banco de Dados
Por motivos de segurança, os dados da conexão com o banco, como usuario e senha, não devem ser commitados. Os dados ficaram armazenados em variaveis chave-valor no arquivo ```.env```, a qual cada um deve criar um para o seu repositorio local, visto que o ```.env``` também vai ser ignorado no commit.
